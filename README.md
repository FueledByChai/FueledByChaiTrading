# Market Connectivity Library

Library providing common functionality for connecting to market data providers and brokers/exchanges.

## Developer Docs

- Shared conventions for exchange modules: `docs/commons-api-conventions.md`
- New exchange scaffolding notes: `scripts/README-new-exchange.md`
- Common-module validator: `scripts/validate-common-module.sh`
- Reference exchange module: `commons/lighter-common-api/README.md`

## Supported Brokers
- Paradex (Crypto DEX)
- Hyperliquid (Crypto DEX) — *partial support*
- Interactive Brokers — *partial support*

## Supported Data Providers
- Paradex (Crypto DEX)
- Hyperliquid (Crypto DEX)
- Interactive Brokers
